# Amanda Massage

Next.js website voor Amanda Massage in Zoetermeer.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Ontwikkelen

```bash
npm install
npm run dev
```

## Belangrijke omgevingsvariabelen

Zie [`.env.example`](/C:/Users/levik/Documents/Playground/.env.example).

- `RESEND_API_KEY`: API key voor het versturen van formuliermails via Resend
- `CONTACT_FROM_EMAIL`: geverifieerd afzenderadres binnen Resend
- `CONTACT_TO_EMAIL`: inbox waar contactaanvragen naartoe gaan
- `NEXT_PUBLIC_ANNOUNCEMENT_ENABLED`: zet tijdelijke banner aan of uit
- `NEXT_PUBLIC_ANNOUNCEMENT_TEXT`: tekst van de tijdelijke banner
